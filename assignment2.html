Welcome to Assignment 2 Tutorial

Follow the step by step instructions to learn how to use Amazon Elastic MapReduce (Amazon EMR). Become a pro in an hour simply by following these steps!


a. Download sample dataset and script
b. Setup Amazon S3 Bucket and upload files
c. Launch Job Flow
d. Analyze Results
e. Have a cup of coffee and marvel at yourself becoming an EMR expert

b. Setup Amazon S3 and upload files

1. Login to Amazon
2. Open Amazon S3 console
3. Click "Create Bucket"
4. Give your bucket a unique name (eg. artist429) and select Singapore in Region dropdown
5. Select the bucket you have just created
6. Click "Create Folder" and create a new folder named "input" 7. Select the folder ("input") you have just created
8. Click "Upload" and select sample data "sampleArtist.csv" to upload
9. Go back one level to your bucket
10. Click "Upload" and select script "wordSplitter.py" to upload
11. Done!

c. Launch Job Flow

1. Open Amazon Elastic MapReduce console
2. Click "Create New Job Flow"
3. Enter a Job Flow Name. (eg. Analyze Artist)
4. Hadoop version*: "Hadoop 1.0.3(Amazon Distribution)"
5. Create a Job Flow*:"Streaming"
6. Continue
7. Input Location*: / (eg. artist429/input)
8. Output Location*: /output (eg. artist429/output)
9. Mapper*: /wordSplitter.py (eg. artist429/wordSplitter.py)
10. Reducer*: aggregate
11. Extra Args*: (leave it blank)
12. Continue
13. Instance Type: Small (m1.small)
14. Core Instance Group:
Instance Count: 2; Instance Type: Small (m1.small)
15. Task Instance Group:
Instance Count: 2; Instance Type: Small (m1.small)
16. Continue
17. Leave all fields on "Advance Options" as defaults
18. Continue
19. Leave all fields on "Bootstrap Actions" as defaults
20. Continue
21. Review information and click "Create Job Flow"
22. Done!

PS: Grab a cup of coffee and wait while EMR does all the work for you!

d. Analyze Results

1. Open Amazon S3 console
2. Navigate to bucket that you have created (eg. artist429)
3. Double click on "output folder"
4. Download all .txt files containing results from the EMR job
5. Done!

e. EMR Expert

Congratulations on becoming an Amazon EMR expert!
