Welcome to Assignment 2 Tutorial

Follow the step by step instructions to learn how to use Amazon Elastic MapReduce (Amazon EMR). Become a pro in an hour simply by following these steps!

1. Download sample dataset and script
2. Setup Amazon S3 Bucket and upload files

2. Setup Amazon S3 and upload files

a. Login to Amazon
b. Open Amazon S3 console
c. Click "Create Bucket"
d. Give your bucket a unique name (eg. artist429) and select Singapore in Region dropdown
e. Select the bucket you have just created
f. Click "Create Folder" and create a new folder named "input" g. Select the folder ("input") you have just created
h. Click "Upload" and select sample data "sampleArtist.csv" to upload
i. Go back one level to your bucket
j. Click "Upload" and select script "wordSplitter.py" to upload
k. Done!

3. Launch Job Flow

a. Open Amazon Elastic MapReduce console
b. Click "Create New Job Flow"
c. Enter a Job Flow Name. (eg. Analyze Artist)
d. Hadoop version*: "Hadoop 1.0.3(Amazon Distribution)"
e. Create a Job Flow*:"Streaming"
f. Continue
g. Input Location*: / (eg. artist429/input)
h. Output Location*: /output (eg. artist429/output)
i. Mapper*: /wordSplitter.py (eg. artist429/wordSplitter.py)
j. Extra Args*: (leave it blank)
k. Continue
l. Instance Type: Small (m1.small)
m. Core Instance Group:
Instance Count: 2; Instance Type: Small (m1.small)
n. Task Instance Group:
Instance Count: 2; Instance Type: Small (m1.small)
o. Continue
p. Leave all fields on "Advance Options" as defaults
q. Continue
r. Leave all fields on "Bootstrap Actions" as defaults
s. Continue
t. Review information and click "Create Job Flow"
u. Done!
